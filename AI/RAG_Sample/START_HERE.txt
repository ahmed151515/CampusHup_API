"Ollama not available"
  → Make sure: ollama serve is running in another terminal

"Index not found"
  → Run: python main.py build-index --data-dir .

"Connection refused"
  → Ollama needs to be running
  → Open new terminal and run: ollama serve

"Slow on first query"
  → Normal! Embedding generation + LLM takes time
  → Subsequent queries are cached and faster

"FAISS error"
  → Run: pip install --upgrade faiss-cpu

🎓 HOW IT WORKS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Your Documents (PDF, Excel)
              ↓
        Document Parser
      [Extract & Split into chunks]
              ↓
        Embeddings Generator
      [all-minilm converts text → vectors]
              ↓
        Faiss Vector Index
      [Fast similarity search on vectors]
              ↓
        RAG Pipeline
      [User query → retrieve similar docs → generate answer]
              ↓
        LLM (llama3.2)
      [Generate human-like response]
              ↓
        Answer to User
      [With source attribution]
