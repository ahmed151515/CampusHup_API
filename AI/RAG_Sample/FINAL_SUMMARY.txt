   from vector_db import FaissVectorDB
   from retrieval import RAGChain
   
   embedder = OllamaEmbedder()
   vector_db = FaissVectorDB()
   vector_db.load()
   rag = RAGChain(vector_db, embedder)
   result = rag.query("Your question")
   print(result["answer"])

❓ FAQ
═══════════════════════════════════════════════════════════════════════════════

Q: Why do I need Ollama running?
A: Ollama provides the embedding model (all-minilm) and LLM (llama3.2)
   locally without needing cloud APIs or internet.

Q: How long does index building take?
A: Depends on document size. ~1-5 minutes for typical documents.
   This is a one-time operation.

Q: Is the index saved?
A: Yes! Faiss index is persisted to disk in models/faiss_index/
   Subsequent queries don't need rebuilding.

Q: Can I add more documents later?
A: Yes. Add files and rebuild index: python main.py build-index --data-dir .

Q: What if embeddings are slow?
A: Normal on first run. Use smaller CHUNK_SIZE or fewer documents.
   Subsequent queries use cached index and are fast.

Q: Can I use different models?
A: Yes. Edit config.py:
   - EMBEDDING_MODEL for better quality/speed tradeoff
   - LLM_MODEL for different answer generation

Q: What if Ollama crashes?
A: Restart it: ollama serve
   The Faiss index is safe and reloadable.
